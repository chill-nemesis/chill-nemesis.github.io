Detailed Description

template<class L, std::enable_if_t< std::is_base_of< PARALLEL::LOOP::Loop, L >::value, int > = 0>
class HORIZON::CORE::APPLICATION::BaseLoopApplication< L, >

Runs a main loop (inside the main thread). Callbacks are to be kept as fast as possible and must not block. There is no guarantee of the execution order of callbacks.
