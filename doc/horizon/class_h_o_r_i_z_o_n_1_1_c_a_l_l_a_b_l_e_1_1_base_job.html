Detailed Description

The base class of all jobs. This is both an interface and a skeleton to prevent code duplication in the template specialisation.

This class is an interface for Job. Its main use is to provide a common type for storing jobs in a container (e.g. in a threadpool implementation, see HORIZON::PARALLEL::ThreadPool).
